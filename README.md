# CivicEye 🏙️  
**Smart Civic Complaint Management System**

CivicEye is a web-based platform designed to streamline the process of reporting, tracking, and resolving civic issues efficiently. It connects citizens with government departments through a structured, role-based system.

---

## 🚀 Features

- 🧑‍🤝‍🧑 Role-based authentication system:
  - Citizen
  - Surveyor
  - Worker
  - Technician
  - Admin

- 📝 Complaint Management:
  - Submit complaints with details & images
  - Track complaint status
  - Categorization by department

- 🏢 Department Handling:
  - Assign complaints to relevant departments
  - Manage department workflows

- 🔄 Assignment System:
  - Assign tasks to workers/technicians
  - Status updates and tracking

- 📊 Analytics Dashboard:
  - Complaint statistics
  - Performance insights

- 🔔 Notifications:
  - Status updates
  - Alerts for assigned tasks

---

## 🛠️ Tech Stack

- **Backend:** Django  
- **Database:** MySQL  
- **Frontend:** HTML, CSS, Bootstrap  
- **Version Control:** Git & GitHub  
- **Project Management:** Jira
